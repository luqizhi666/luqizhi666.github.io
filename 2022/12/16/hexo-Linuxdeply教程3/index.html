Hexo 配置

贴上本站配置，仅供参考

Fluid

npm install --save hexo-theme-fluid

创建 _config.fluid.yml，将主题的 _config.yml 内容复制进去。

hexo new page about

创建成功后，编辑博客目录下 /source/about/index.md，添加 layout 属性。

修改后的文件示例如下：

`

title: about

layout: about

—`
